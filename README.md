# egao-css

A CSS utility to create Kaomoji.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the CSS:
   ```bash
   npm run build
   ```
3. Use `dist/main.css` in your HTML.

## Development

- Add utility classes to `src/main.css`.
- Build with `npm run build`.
